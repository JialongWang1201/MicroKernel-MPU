# Security Policy

  ## Supported Versions

  This project currently provides security updates for:

  | Version | Supported |
  | ------- | --------- |
  | `main`  | Yes       |
  | older commits / unreleased snapshots | No |

  ## Reporting a Vulnerability

  Please do not report security vulnerabilities through public GitHub issues.

  Use one of these channels instead:

  - GitHub Private Vulnerability Reporting for this repository, if enabled
  - Contact the maintainer directly through the repository owner account

  When reporting, include:

  - affected commit or branch
  - reproduction steps
  - impact assessment
  - any suggested fix or mitigation

  You can expect an initial acknowledgement within 7 days.
  If the report is accepted, a fix will be prepared and disclosed after remediation is available.
